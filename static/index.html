🏥 VoiceClinicAgent API

OpenEnv-compatible RL environment for clinic appointment booking agents

⚙️ Configuration Required

Note: This Space requires environment variables to be configured in the Space settings:

  • API_BASE_URL - OpenAI API base URL (e.g., https://api.openai.com/v1)
  • MODEL_NAME - Model name (e.g., gpt-3.5-turbo)
  • HF_TOKEN - Hugging Face token or OpenAI API key

Without these variables, the /reset endpoint will return an error.

📚 API Documentation

Interactive API documentation: /docs

🔗 Available Endpoints

GET /health

Health check endpoint

POST /reset

Initialize a new episode with a task_id (easy_001, medium_001, hard_001)

POST /step

Execute one environment step with an action

GET /state/{episode_id}

Get current episode state including final score when complete

🎯 Available Tasks (9 Total)

Easy (15 turns each)

  • easy_001 - Adult patient, clear symptoms, straightforward booking
  • easy_002 - Simple appointment request with basic questions
  • easy_003 - Direct booking scenario with minimal complexity

Medium (20 turns each)

  • medium_001 - Parent booking for child, mild vagueness
  • medium_002 - Insurance verification required, moderate complexity
  • medium_003 - Multiple department options, preference handling

Hard (25 turns each)

  • hard_001 - Limited slots, urgent cases, mixed Hindi-English
  • hard_002 - High-risk patient, complex medical history
  • hard_003 - Resource pressure, conflicting information, privacy challenges

📖 Quick Start

Example: Start a new episode

curl -X POST "https://huggingface.co/spaces/Kabs-123/voice-clinic-agent/reset" \
  -H "Content-Type: application/json" \
  -d '{"task_id": "easy_001", "seed": 42}'
