🏥 VoiceClinicAgent API

OpenEnv-compatible RL environment for clinic appointment booking agents

📚 API Documentation

Interactive API documentation: /docs

🔗 Available Endpoints

GET /health

Health check endpoint

POST /reset

Initialize a new episode with a task_id (easy_001, medium_001, hard_001)

POST /step

Execute one environment step with an action

GET /state/{episode_id}

Get current episode state including final score when complete

🎯 Available Tasks

  • easy_001 - Adult patient, clear symptoms, straightforward booking (15 turns)
  • medium_001 - Parent booking for child, mild vagueness (20 turns)
  • hard_001 - Limited slots, urgent cases, mixed language (25 turns)

📖 Quick Start

Example: Start a new episode

curl -X POST "https://huggingface.co/spaces/Kabs-123/voice-clinic-agent/reset" \
  -H "Content-Type: application/json" \
  -d '{"task_id": "easy_001", "seed": 42}'
